## 📌 Project Overview

ASCEND 2.0 is a fully functional frontend eCommerce platform focused on **fitness and gym equipment**.

It includes:
- Product listing (50+ items)
- Cart system
- Wishlist system
- Billing & checkout page
- Account management system

The design is inspired by **modern SaaS dashboards + gaming UI + cyberpunk aesthetics**.

---

## ⚡ Key Features

### 🛒 Equipment Store
- 50+ gym products (Dumbbells, Barbells, Machines, etc.)
- Categories & filters
- Search functionality
- “Best” & “New” product tags  

---

### 🛍️ Cart System
- Add/remove products  
- Quantity management  
- Auto price calculation  
- LocalStorage-based persistence

### ❤️ Wishlist System
- Save favorite products

## 🧠 How It Works

### 🧾 Product System
- All products are stored in `data.js`
- Rendered dynamically using JavaScript

### 🛒 Cart Logic
- Stored in `localStorage`
- Functions:
  - `addToCart()`
  - `removeFromCart()`
  - `updateCartQty()`

### ❤️ Wishlist Logic
- Toggle-based system  
- Stored in localStorage  

### 🔁 Shared UI
- Navbar and footer are injected dynamically across all pages
